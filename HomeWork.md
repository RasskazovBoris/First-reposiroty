# Основные команды Git

## 1. git init

Инициализация репозитория 

## 2. git add

Добавление файлов

## 3. git status

Проверка статуса репозитория

## 4. git log

Просмотр истории коммитов

## 5. git branch

Просмотр списка всех веток

## 6. git branch name

Создание новой ветки с именем name

## 7. git merge

Слияние веток
